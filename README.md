Intelligent Website Change Detection System

A smart web monitoring tool that tracks changes in websites and notifies users in real-time.

Features
Enter any website URL to monitor
Detects changes using content comparison
Highlights differences (added / removed text)
Save favorite websites for tracking
Real-time browser notifications on updates
Maintains version history of websites
Tech Stack

Frontend: React.js
Backend: Node.js + Express
Libraries: Axios, Cheerio, Diff
Deployment: Vercel (Frontend), Render (Backend)

⚙️ How It Works
User enters a website URL
Backend fetches and extracts page content
Previous and current versions are compared
Differences are highlighted using diff algorithm
Notifications are triggered when changes are detected
